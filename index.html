Albox

Diaporama simple pour visionner ses photos

View project onGitHub

Introduction

Cet outil permet de visionner ses images présent sur vos sites perso tel que free.fr ou chez ovh.

Condition requise

Aucune connaissance technique n'est nécessaire, il faut juste savoir donner des droits à un dossier (expliqué au dessous). Vous devez toutefois avec la librairie GD installé sur votre serveur web (demandé à l'administrateur de votre site)

Installation du paquet.

Décompressez le zip (si vous avez choisie l'archive) dans votre répertoire où sont les photos.

Autorisation de droit

Donnez les droits au dossier /albox/images afin de pouvoir générer les miniatures. Pour se faire, via ligne de commande :

... chmod 777 -Rf albox/images ...

Sinon, avec votre client FTP, clique droit sur le répertoire albox/images puis "Droits d'accès au fichier ...". Mettre 777 (ou tout cocher) puis ok ajout droit

Configuration du parameters.ini

Vous avez la possibilité de configurer votre albox avec l'aide du parameters.ini. Il suffit de l'ouvrir avec un editeur de texte, comme notepad, puis de changer les paramètres.

Paramètre disponible :

  • [title] : Titre du diaporama
  • [resizeFolder] : si vous souhaitez changer le dossier de destination des miniatures
  • [excludeFolder] : liste les dossiers qui ne seront pas affiché par albox
  • [allowFormat] : liste des formats d'image autorisé
  • [sizeImage] : taille des miniatures générées.
